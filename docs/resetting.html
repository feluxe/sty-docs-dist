Resetting

Sty uses ANSI SGR parameters to reset previously applied styles. More about ANSI Select Graphic Rendition (SGR) on wikipedia: ANSI SGR

Sty’s default reset register

These are the default attributes for the sty.rs register-object:

Attribute

Default Renderer
